# FinTrack: Digital Wallet Analytics & Business Intelligence Platform

FinTrack is an end-to-end fintech data analytics project built to simulate the work of a Data Analyst in a digital wallet company.

The project analyzes thousands of digital wallet transactions to uncover customer behavior, transaction performance, merchant trends, and business opportunities through data-driven insights.

The workflow includes data cleaning, exploratory data analysis (EDA), SQL business analysis, and interactive Power BI dashboards to transform raw transaction data into actionable recommendations.

## Objectives

* Clean and validate raw transaction data.
* Analyze transaction patterns and customer behavior.
* Evaluate product and merchant performance.
* Measure transaction success rates and operational efficiency.
* Assess the impact of cashback and loyalty programs.
* Build dashboards to communicate key business insights.

## Tech Stack

* Python (Pandas, NumPy, Matplotlib)
* SQL
* Power BI
* Git & GitHub

## Project Workflow

Raw Dataset → Python Data Cleaning → Exploratory Data Analysis → SQL Business Analysis → Power BI Dashboard → Business Recommendations
